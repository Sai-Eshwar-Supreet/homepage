# homepage

A responsive homepage layout built with HTML and CSS.  
The project recreates a typical portfolio-style landing page with a hero section, project showcase, and contact section.

## Live Preview

[View on GitHub Pages](https://sai-eshwar-supreet.github.io/homepage/)

## Tech

- HTML
- CSS

## Run Locally

Clone the repo and open `index.html` in a browser.
No build tools or dependencies are required.

```bash
git clone https://github.com/Sai-Eshwar-Supreet/homepage.git
```
